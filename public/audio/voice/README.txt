PIP / ApotheCorp voice lines
============================

Drop mp3 recordings here with the exact filenames listed below. Each file is
optional — missing files are silently skipped at runtime, so you can record
and ship these in any order.

See docs/VOICE_BIBLE.md for:
  - The ElevenLabs voice prompt + 60-second clone-sample script
  - Full spoken text for each line
  - Delivery tags (tone / cadence / splice beats)

Recording spec:
  - Mono, 44.1 kHz, mp3, 128 kbps is plenty
  - Leave ~50 ms of head/tail silence
  - Keep levels consistent; the VoiceManager mixes them at a single volume

The authoritative list of filenames lives in src/audio/voiceLines.js. The
list below is a snapshot for quick reference.
